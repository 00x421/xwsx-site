## 开发工具

### VS Code

依然是主力编辑器，配合以下插件：

- **GitLens** — Git 历史可视化
- **Error Lens** — 行内错误提示
- **Tailwind CSS IntelliSense** — 类名补全

### Warp

终端工具，AI 辅助命令补全很好用，特别是记不住复杂命令的时候。

## 写作工具

### Obsidian

本地笔记 + 知识库管理，双向链接让笔记之间产生连接，长期积累下来价值很大。

### Notion

团队协作和项目管理用 Notion，数据库视图灵活，适合做内容规划。

## 任务管理

### Todoist

GTD 理念的待办事项工具，跨平台同步快，自然语言输入日期很方便。

## 小结

工具只是手段，关键是找到适合自己的工作流。以上是我过去一年沉淀下来的工具组合，供参考。
